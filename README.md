# hackercoop.is

website for the [Hacker Coop](http://hackercoop.is) built using [DocPad](http://docpad.org).

## how to

### install

```
git clone https://github.com/holonomy/holonomy.is
npm install
```

### run

`npm start`

### develop

`npm run develop`

### deploy

`npm run deploy`
